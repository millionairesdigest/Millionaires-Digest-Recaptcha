== Description ==

Allow the Millionaire's Digest to add reCAPTCHAs on login, registration, reset password pages, comment forms, and any other pages set and defined by the Millionaire's Digest with special thanks and all credits going to Carlos E. G. Barbosa for the original plugin.

== Installation ==

1. Visit Dashboard-> Plugins-> Add New
2. Install plugin
3. Activate
